# Projeto Fila de Clientes

Este é um projeto de estudo de **Estruturas de Dados** em Java, com foco em **Fila (FIFO - First In First Out)**.

## Estruturas Implementadas

- **Cliente**: representa um cliente com nome e plano.
- **Fila**: estrutura de dados que organiza a ordem de atendimento.
- **Lista** (internamente usada na Fila).

## Funcionalidades

- Adicionar clientes na fila (`enfileirar`)
- Atender clientes na ordem de chegada (`desenfileirar`)
- Visualizar a fila atual

## Como Rodar

1. Abra o projeto no VS Code
2. Compile e rode a classe `Main.java`
3. Observe a simulação de chegada e atendimento de clientes

## Observações

- O foco do projeto é demonstrar a implementação correta da **estrutura de fila**.
- Simulações de chegadas e saídas podem ser adaptadas para testar diferentes cenários.
